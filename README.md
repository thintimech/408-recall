# 408 Recall

408 Recall 是一个面向计算机考研 408 的本地优先主动回忆工具。它帮助用户围绕数据结构、计算机组成原理、操作系统、计算机网络创建记忆卡片，并通过今日复习队列记录掌握情况。

## 项目边界

- 不是题库；
- 不内置商业 PDF、辅导书原文或课程资料；
- 不提供真题解析库；
- 不做账号系统、后端服务、多设备同步；
- MVP 不接入 AI，后续如接入也只作为辅助制卡工具。

## MVP 功能

- 内置 408 四门课基础章节树；
- 新增、编辑、删除知识点节点；
- 新增、编辑、删除记忆卡片；
- 新卡片自动进入今日复习；
- 主动回忆：先看正面，再显示答案；
- 复习结果：忘了、困难、记得、熟练；
- 自动计算下次复习日期；
- 查看复习记录；
- JSON 导出和覆盖导入。

## 本地开发

```bash
npm install
npm run dev
```

## 验证

```bash
npm run type-check
npm run build
```

## 数据说明

用户数据保存在当前浏览器的 IndexedDB 中。清理浏览器数据可能导致卡片和复习记录丢失，请定期在设置页导出 JSON 备份。

JSON 导入采用覆盖模式：导入前会清空当前本地数据，然后写入备份文件中的数据。
